# Smart Bank Management System

## Project Description

Smart Bank Management System is a console-based Core Java application that simulates basic banking operations. It allows users to register customers, create bank accounts, perform transactions, and manage customer information using Object-Oriented Programming concepts.

---

## Technologies Used

- Core Java
- Object-Oriented Programming (OOP)
- Java Collections (ArrayList)
- Exception Handling
- Java Time API
- IntelliJ IDEA

---

## Features

- Customer Registration
- View All Customers
- Search Customer
- Update Customer Details
- Delete Customer
- Create Savings/Current Account
- Auto Generate Account Number
- Login using Account Number and PIN
- Deposit Money
- Withdraw Money
- Transfer Money
- Check Account Balance
- Change PIN
- View Transaction History

## How to Run

1. Open the project in IntelliJ IDEA.
2. Build the project.
3. Run `Main.java`.
4. Select options from the menu to perform banking operations.
